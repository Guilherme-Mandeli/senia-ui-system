Escala de Breakpoints

Puntos de control responsivos preconfigurados en Sènia UI:

xs (Mobile) 480px --breakpoint-xs
sm (Tablet Small) 768px --breakpoint-sm
md (Tablet / Laptop) 1024px --breakpoint-md
lg (Desktop) 1280px --breakpoint-lg
xl (Wide Desktop) 1440px --breakpoint-xl
2xl (Ultra Wide) 1920px --breakpoint-2xl

Uso en CSS Nativo (Media Queries)

Uso recomendado en tus archivos de estilos para maquetaciones responsivas:

/* Mobile First */

/* Tablet Small */
@media (min-width: 768px) {
  .mi-bloque {
    /* sm */
  }
}

/* Tablet / Laptop */
@media (min-width: 1024px) {
  .mi-bloque {
    /* md */
  }
}

/* Desktop */
@media (min-width: 1280px) {
  .mi-bloque {
    /* lg */
  }
}

/* Wide Desktop */
@media (min-width: 1440px) {
  .mi-bloque {
    /* xl */
  }
}

/* Ultra Wide */
@media (min-width: 1920px) {
  .mi-bloque {
    /* 2xl */
  }
}
