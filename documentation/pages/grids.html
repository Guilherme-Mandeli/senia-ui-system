Autoadaptative Layout

Grilla Autoadaptativa (`.l-grid--auto`)

Una grilla inteligente inspirada en PicoCSS que distribuye automáticamente columnas del mismo tamaño sin requerir clases de span numéricas. Se apila en mobile (1 columna) y se expande en tablets y desktop de forma fluida.

Best Practices

Recomendaciones

Use layouts predefinidos

Prefiera .l-col-* para páginas simples, contenido editorial y layouts rápidos.

Use la grilla atómica

Utilice .u-grid-cols-* y spans para dashboards, interfaces complejas y composiciones avanzadas.

Mantenga spacing consistente

Combine layouts con utilidades .u-gap-* y paddings coherentes.

Diseñe Mobile First

Las grillas colapsan naturalmente en pantallas pequeñas para maximizar legibilidad.
