Navegación y Breadcrumbs

Menús estructurales flexibles de navegación (`nav`) y rutas de migas de pan (`breadcrumbs`) para gestionar la orientación espacial del usuario.

Barra de Navegación Estándar (`nav` / `.c-nav`)

Un contenedor flex alineado que distribuye el logo y los enlaces de navegación de forma automatizada:

Navegación Superior
<nav> .c-nav <a>[aria-current="page"]
  • Senia UI
  • Proyectos
  • Servicios
  • Contacto

Migas de Pan Nativas (`nav[aria-label="breadcrumb"]`)

Migas de pan estructuradas de forma semántica con separadores automáticos para páginas internas:

Migas de Pan (Breadcrumbs)
<nav>[aria-label="breadcrumb"] .c-breadcrumb
  • Inicio
  • Documentación
  • Componentes
  • Navegación

Código de Ejemplo

<!-- Barra de navegación -->
<nav>
  <ul>
    <li><strong>Mi Proyecto</strong></li>
  </ul>
  <ul>
    <li><a href="#" aria-current="page">Inicio</a></li>
    <li><a href="#">Servicios</a></li>
    <li><button class="c-btn c-btn--sm">Salir</button></li>
  </ul>
</nav>

<!-- Migas de pan (Breadcrumbs) -->
<nav aria-label="breadcrumb">
  <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Panel</a></li>
    <li>Detalle del Item</li>
  </ul>
</nav>
