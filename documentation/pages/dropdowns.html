Menús Desplegables (Dropdowns)

Menús de opciones flotantes autocontenidos y libres de JavaScript, maquetados utilizando la estructura HTML nativa de acordeones.

Dropdown Classless Semántico

Al utilizar selectores semánticos como details.dropdown, la lista interna se posiciona y oculta de forma automática. El menú se abre al hacer clic y se cierra al hacer clic fuera o re-seleccionar.

Dropdown Semántico
<details>.dropdown <summary> <ul>
Opciones de Usuario
  • Ver Perfil
  • Configuración
  • Seguridad
  • Cerrar Sesión
Idiomas
  • Español (ES)
  • English (EN)
  • Português (PT)

Dropdown en BEM (`.c-dropdown`)

Variación con clases explícitas bajo nomenclatura BEM para mayor consistencia de marcado en arquitecturas de componentes modulares:

Dropdown en Clases BEM
.c-dropdown .c-dropdown__menu
Acciones Rápidas
  • Exportar PDF
  • Descargar Excel
  • Imprimir Reporte

Código de Ejemplo

<!-- Dropdown Classless Semántico -->
<details class="dropdown">
  <summary>Menú de Opciones</summary>
  <ul>
    <li><a href="#">Opción 1</a></li>
    <li><a href="#">Opción 2</a></li>
  </ul>
</details>

<!-- Dropdown BEM con botón contorneado -->
<details class="c-dropdown">
  <summary class="c-btn c-btn--outline">Acciones</summary>
  <ul class="c-dropdown__menu">
    <li><a href="#">Editar</a></li>
    <li><a href="#">Eliminar</a></li>
  </ul>
</details>
