Acordeones y Colapsables (`details`)

Paneles desplegables construidos sobre la etiqueta semántica nativa <details> para organizar información jerárquica o secundaria de forma limpia.

Acordeón Estándar (Card-Style)

Variante nativa por defecto. Los elementos se enmarcan con bordes, sombras sutiles y fondo de tarjeta, óptimos para secciones aisladas como FAQs.

Etiquetas details / summary
<details> <summary>
¿Qué tecnologías utiliza el framework Senia UI?

Senia UI está desarrollado íntegramente con CSS Vanilla moderno, haciendo uso de variables personalizadas (CSS Custom Properties) para la gestión cromática y de layout, y de Cascading Layers (@layer) para resolver la especificidad y precedencia de estilos.

¿Cómo se gestiona el cambio de color para el modo oscuro?

El framework emplea la clase global .theme-dark en el elemento raíz del documento. Todas las variables de color están programadas en formato HSL, permitiendo que al activar esta clase se re-mapeen automáticamente todos los tonos y contrastes en tiempo de ejecución.

Variante Limpia / Minimalista (`.c-accordion--clean`)

Utilizando el modificador BEM .c-accordion--clean, se remueven los bordes circundantes, fondos y sombras, dejando únicamente una línea divisoria inferior y un estilo tipográfico más integrado.

Especificaciones técnicas de compilación

El flujo de compilación corre bajo PostCSS acoplado a un script en Node.js que procesa:

  • postcss-import: Resolución de importaciones.
  • postcss-nesting: Anidación nativa de reglas.
  • autoprefixer: Adición de prefijos de compatibilidad.
  • cssnano: Minificación extrema para producción.
