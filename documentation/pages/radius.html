Tokens de Diseño

Bordes & Radios

Escala geométrica estandarizada para esquinas curvas de contenedores, inputs, botones y modales.

Muestra de Bordes

Deslice o interactúe con los contenedores para apreciar las distintas curvaturas y esquinas redondeadas:

--radius-none 0px
--radius-i 3px
--radius-ii 4px
--radius-iii 6px
--radius-iv 8px
--radius-v 10px
--radius-vi 12px
--radius-vii 16px
--radius-full 9999px

Valores y Convenciones de Uso

Token CSS Valor en Pixeles Uso Recomendado en Sènia
--radius-none 0px Esquinas en ángulo recto. Ideal para elementos adjuntos o extendidos a pantalla completa.
--radius-i 3px Redondeado sutil para pequeños checkboxes, tooltips o selectores mínimos.
--radius-ii 4px Botones pequeños, mini badges, campos de texto compactos y tags simples.
--radius-iii 6px Botones por defecto, inputs regulares y selectores estándar en Sènia UI.
--radius-iv 8px Curvatura por defecto para tarjetas (.c-card), layouts fijos e inputs grandes.
--radius-v 10px Paneles informativos destacados, notificaciones toast y popovers dinámicos.
--radius-vi 12px Modales del sistema (.c-modal), diálogos de confirmación y cards muy amplias.
--radius-vii 16px Grandes contenedores hero, banners promocionales y elementos gráficos destacados.
--radius-full 9999px Badges tipo píldora (.c-badge--pill), avatares circulares y botones redondos (FAB).
