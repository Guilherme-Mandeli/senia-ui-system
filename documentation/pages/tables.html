Tablas de Datos (Tables)

Tablas estructuradas para representar listados masivos de datos de forma limpia, responsiva y contrastada.

Tabla Cebrada e Interactiva

Las clases modificadoras .c-table--striped y .c-table--hover colorean de forma alterna las filas del listado e iluminan la celda seleccionada al pasar el cursor (hover) para mejorar el escaneo de grandes columnas de datos.

Responsividad Horizontal (.c-table-container)

El contenedor inteligente .c-table-container aplica un scroll horizontal responsivo en pantallas móviles pequeñas para evitar la rotura o desborde del layout global de la página.

Redimensiona tu navegador para simular una resolución de smartphone de menos de 480px de ancho y prueba a deslizar horizontalmente la tabla de arriba.
