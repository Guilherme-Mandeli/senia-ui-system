Estructura de Clases

Para aplicar los colores, utiliza el formato .u-text-[paleta]-[escala], donde paleta es primary, secondary, neutral, info, success, warning o error y la escala abarca los niveles 50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950 (ejemplo: .u-text-primary-500).

Temas de Contraste Activo (Light & Dark Contrast)

Clases de utilidad global para conmutar el color de primer plano a claro u oscuro. Adicionalmente, estas clases propagan de forma inteligente el contraste hacia elementos secundarios como iconos SVG, flechas indicadoras y efectos de carga interactivos.

u-contrast-light

Contraste Claro

Fuerza al texto de todo el contenedor a renderizarse en color blanco puro. Los iconos vectoriales adyacentes y cargadores de carga heredan el tono automáticamente.

u-contrast-dark

Contraste Oscuro

Fuerza al texto del contenedor a color gris oscuro/negro corporativo. Se adapta dinámicamente a fondos claros (como amarillo primario o arena).
