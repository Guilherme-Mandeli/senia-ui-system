Especificación de Tiempos y Curvas

Token CSS Especificación Técnica Propósito Principal
--transition-duration-fast 150ms Micro-interacciones ultrarrápidas: hovers de botones, cambios de iconos y links.
--transition-duration-normal 250ms Transición intermedia óptima: elevaciones de tarjetas y aperturas de barras laterales.
--transition-duration-slow 400ms Transiciones lentas y majestuosas: modales fluidos, diálogos grandes y banners rotativos.
--transition-duration-extra-slow 720ms Transición muy lenta y deliberada: transiciones detalladas, layouts complejos y secuencias animadas pausadas.
--transition-easing-linear linear Transición con velocidad constante. Utilizado únicamente para fundidos de opacidad y color.
--transition-easing-premium cubic-bezier(0.2, 0, 0, 1) Curva de aceleración ultra fluida inspirada en interfaces nativas de Apple y Google.
