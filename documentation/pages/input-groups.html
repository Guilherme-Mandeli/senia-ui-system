Grupos de Entrada (Input Groups)

Combinaciones compactas de campos de entrada de texto, selectores y botones de acción unificados en un único control visual continuo.

Grupo Estándar (`.c-input-group` o `[role="group"]`)

Permite fusionar de forma fluida bordes, desactivar radios de esquina internos y superponer bordes adyacentes para que se comporten como un solo elemento unificado:

Campos Combinados Estándar
.c-input-group [role="group"]

Búsqueda Redondeada (`.c-input-group--search` o `[role="search"]`)

Aplica radios de borde totalmente circulares (estilo píldora/cápsula) en los extremos exteriores de los campos agrupados para crear una barra de búsqueda premium:

Barra de Búsqueda de Esquinas Redondas
.c-input-group--search [role="search"]

Código de Ejemplo

<!-- Grupo estándar (input + botón) -->
<div class="c-input-group">
  <input type="text" class="c-input" placeholder="Buscar...">
  <button class="c-btn c-btn--primary">Acción</button>
</div>

<!-- Variante de barra de búsqueda redondeada -->
<div class="c-input-group--search">
  <input type="text" class="c-input" placeholder="Escribe algo...">
  <button class="c-btn c-btn--primary">Buscar</button>
</div>
