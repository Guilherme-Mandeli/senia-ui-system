Personalización de Temas (Highlight & Accents)

Aprende a personalizar el color de acento y resaltar toda la web de forma simple mediante variables de configuración CSS y fallbacks neutros de alta accesibilidad.

Filosofía de Variables de Configuración

Senia UI libera variables personalizadas específicas que actúan como la "semilla cromática" del sitio web. Por defecto, si el desarrollador no define estas variables, el framework utiliza **colores neutros de alto contraste** (negro en modo claro y blanco en modo oscuro) garantizando una legibilidad óptima out-of-the-box.

Regla de Oro de Accesibilidad y Control

Importante: El framework **nunca utiliza los colores primarios y secundarios corporativos como fallback por defecto**. Dado que la paleta primaria (ej. Amarillo Senia) puede ser muy clara y comprometer el contraste WCAG en fondos claros, los fallbacks por defecto están vinculados estrictamente a la paleta neutra (Gris muy oscuro var(--color-neutral-900) en modo claro, y Gris muy claro var(--color-neutral-100) en modo oscuro) sobre los cuales tenemos control absoluto de legibilidad.

Cómo Integrarlo en tu Proyecto

Para personalizar el aspecto cromático del framework en tu sitio web, simplemente declara las variables de configuración en tu hoja de estilos principal dentro del selector :root y de la clase .theme-dark: 

/* Variables configuradas por el desarrollador */
:root {
  --senia-primary-color: #1e40af;       /* Highlight en modo claro */
  --senia-primary-hover: #1d4ed8;       /* Hover en modo claro */
  --senia-primary-light: #eff6ff;       /* Alertas o fondos suaves */
  --senia-primary-contrast: #ffffff;    /* Texto sobre fondo highlight */

  /* Opcional: Si el acento es oscuro, fuerza el checkmark/radio de los inputs a blanco */
  --senia-svg-checkmark: var(--svg-checkmark-light);
  --senia-svg-radio: var(--svg-radio-light);
}

/* Sobrescribir los acentos solo para modo oscuro */
.theme-dark {
  --senia-primary-color-dark: #60a5fa;  /* Highlight en modo oscuro */
  --senia-primary-hover-dark: #93c5fd;  /* Hover en modo oscuro */
  --senia-primary-light-dark: #1e293b;  
  --senia-primary-contrast-dark: #0f172a; /* Texto sobre highlight oscuro */

  /* Opcional: Si en modo oscuro prefieres checkmark oscuro */
  --senia-svg-checkmark: var(--svg-checkmark-dark);
  --senia-svg-radio: var(--svg-radio-dark);
}

Ubicación y Variables Clave

Las variables de estilos base y temas se encuentran organizadas en los siguientes ficheros dentro de la estructura de tokens:

  • src/tokens/colors.css: Define la paleta de colores global y los valores por defecto del sistema.
  • src/themes/light.css: Mapea los colores semánticos, fondos e inputs del tema claro.
  • src/themes/dark.css: Mapea los colores semánticos, fondos e inputs del tema oscuro.

Variables más propensas a actualizarse

Estas variables representan la personalización principal de tu marca:

Variable Uso y Descripción
--senia-primary-color Color de acento principal en modo claro. Define botones primarios, enlaces activos y focos.
--senia-primary-contrast Color del texto sobre el fondo de acento en modo claro (ej. #ffffff o #111827).
--senia-primary-color-dark Color de acento principal en modo oscuro.
--senia-primary-contrast-dark Color del texto sobre el fondo de acento en modo oscuro.
--senia-svg-checkmark Define el color del checkmark activo (var(--svg-checkmark-light) o var(--svg-checkmark-dark)).
