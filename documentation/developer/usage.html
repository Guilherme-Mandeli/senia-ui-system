Integración & Uso de Estilos

Aprende a integrar, estructurar y extender los estilos de Senia UI CSS Framework en tus aplicaciones web con un alto rendimiento.

1. Cómo integrar el Framework

Para hacer uso del framework en cualquier proyecto del grupo, tienes dos vías principales de integración:

Recomendado

Vía Compilada (Producción / CDN)

Una vez construido el framework (npm run build), se generan dos archivos CSS independientes y sus versiones minificadas en el directorio /dist/:

1. Estilo Esencial (Tokens, Temas y Reset):

Perfecto para usar solo variables y estilos base nativos.

<link rel="stylesheet" href="https://cdn.grupo-senia.com/assets/senia-essential.min.css">
2. Suite Completa (Componentes, Layouts y Utilidades):

Nuestra suite completa lista para construir interfaces estilo Bootstrap.

<link rel="stylesheet" href="https://cdn.grupo-senia.com/assets/senia-full.min.css">
Entorno Local

Vía PostCSS / Módulos

Si estás desarrollando dentro del repositorio de Senia UI o en un bundle de Vite con PostCSS, puedes importar cualquiera de los dos entrypoints globales:

1. Estilo Esencial: 
@import "senia-ui-system/src/essential.css";
2. Suite Completa: 
@import "senia-ui-system/src/style.css";

2. Convenciones de Nomenclatura

Utilizamos una metodología basada en **BEM Simplificado** combinada con prefijos semánticos para evitar conflictos y asegurar que las clases se auto-expliquen:

Prefijo Propósito Ejemplo de Clase Descripción
c- Componente encapsulado .c-button, .c-card Bloque estilizado con comportamiento visual autónomo.
l- Layout & Estructura .l-container, .l-grid Define contenedores de rejilla, layouts flexibles u organizadores espaciales.
u- Utility (Clases Atómicas) .u-mb-vi, .u-text-center Estilos atómicos de única responsabilidad para retoques de margen, padding, alineamiento, etc.

3. Personalización y Temas

Dado que todo el framework utiliza **Variables CSS Nativas**, es extremadamente fácil sobrescribir los valores por defecto (Tokens) en un componente o sección en particular sin necesidad de recompilar el CSS: 

/* Sobrescribir el color primario de Senia UI solo dentro de un bloque específico */
.mi-seccion-personalizada {
  --color-primary-500: 210 100% 56%; /* Cambia a HSL de azul brillante */
  --radius-iv: 0px; /* Bordes totalmente rectos */
}
