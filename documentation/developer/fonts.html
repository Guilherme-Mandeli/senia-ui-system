Lato & Gill Sans

Uso de Fuentes & Jerarquía del CDN

Directrices para el consumo y optimización de las dos fuentes del sistema: Lato (contexto serio por defecto) y Gill Sans (diseño general corporativo).

1. Optimización del CDN: Carga bajo demanda (Lazy Loading)

Para garantizar velocidades de carga óptimas de 100% y excelentes puntuaciones de Core Web Vitals (LCP), nuestro framework registra las familias tipográficas utilizando reglas `@font-face` dinámicas.

🔒 Sin Descarga Forzada

Los navegadores modernos implementan una optimización automática nativa: **solo realizan la petición de descarga (HTTP request) de los archivos de fuentes si la página actual tiene al menos un nodo en el DOM que utilice de manera explícita ese peso de fuente y familia en particular.**

Por tanto, al usar por defecto únicamente los pesos `400` y `700` en tu layout, las variantes de pesos extremos (`Light 300` o `Heavy 900`) **nunca consumirán ancho de banda de tu usuario**, optimizando al máximo el rendimiento móvil y de escritorio.

2. Mapeo de Pesos Tipográficos (Lato & Gill Sans)

Usa la siguiente tabla para aplicar los pesos exactos y evitar que el navegador sintetice negritas o cursivas deformadas de forma artificial:

Familia / Peso CSS Font Weight Token Recomendado Área de Uso Común
Lato Light / Gill Sans Light 300 --font-weight-300 Subtítulos descriptivos de gran tamaño.
Lato Regular / Gill Sans Regular 400 --font-weight-400 Texto principal de lectura (body) e interfaces de usuario.
Lato Medium / Gill Sans Medium 500 --font-weight-500 Botones, menús de navegación secundarios, texto interactivo.
Gill Sans Semibold 600 --font-weight-600 Títulos por defecto de Gill Sans.
Lato Bold / Gill Sans Bold 700 --font-weight-700 Títulos destacados de Lato, etiquetas semánticas.
Lato Black / Gill Sans Heavy 900 - Encabezados ultra-destacados (Hero Sections).

3. Ejemplos de Implementación en Código CSS

Aquí tienes fragmentos listos para copiar y usar en tus componentes, garantizando que el navegador cargue el peso y estilo exactos de tu CDN: 

/* 1. Usar el peso regular por defecto para textos en contexto Lato (Serio) */
.mi-parrafo-formal {
  font-family: var(--font-family-lato);
  font-weight: var(--font-weight-400); /* Usa Lato Regular (400) */
}

/* 2. Usar un título formal destacado en Lato */
.mi-titulo-formal {
  font-family: var(--font-family-lato);
  font-weight: var(--font-weight-700); /* Usa Lato Bold (700) */
}

/* 3. Invocar un peso pesado (Heavy) en Gill Sans para branding */
.banner-destacado-marca {
  font-family: var(--font-family-gill-sans);
  font-weight: 900; /* Usa Gill Sans Heavy */
}
